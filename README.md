# Unipass Manager API

Sistema de gestão para administradores do UniPass.

## Pré-requisitos

- Java 17 ou superior
- Maven (ou usar o wrapper incluído)

## Como executar

1. Clone o repositório
2. Navegue até a pasta do projeto
3. Execute o comando:

Linux:
```bash
./mvnw spring-boot:run
```
Windows:
```bash
mvnw.cmd spring-boot:run
```
